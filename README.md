# IPL Analysis Dashboard

## Project Overview
This project analyzes Indian Premier League (IPL) data using Python. The objective is to uncover insights related to team performance, player statistics, match outcomes, and season trends through data visualization and exploratory data analysis.

## Technologies Used
- Python
- Pandas
- NumPy
- Plotly
- Jupyter Notebook

## Features

- Analysis of total matches won by each IPL team in the 2022 season.
- Comparison of matches won while defending a target versus chasing a target.
- Identification of the best Player of the Match performers.
- Analysis of the leading bowlers based on wickets and bowling performance.
- Identification of players with the highest number of Player of the Match awards.
- Analysis of the top run scorers in IPL 2022.
- Team-wise performance analysis and win distribution.
- Interactive data visualizations using Plotly.
- Exploratory Data Analysis (EDA) on IPL 2022 match data.
- Cricket insights generated from historical IPL statistics.

## Conclusion
The analysis provides valuable insights into IPL team strategies, player performances, and match-winning factors using historical IPL data.
